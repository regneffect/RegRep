# Reggie Obsidian + Claude OS

This vault is designed around one principle:

**Capture first. Sort later.**

Reggie should not have to maintain a perfect productivity system manually. The system should survive bad days, busy days, and overloaded weeks.

## Daily Use

1. Put everything into `00_Capture/Capture.md`.
2. Add Claude tasks to `05_Queue/Claude Queue.md`.
3. Run Morning Briefing.
4. Run Capture Processor.
5. Let Claude sort, summarize, and route information.

## Weekly Use

Every Monday:
1. Update the Weekly Priorities section in `Claude.md`.
2. Run Morning Briefing.
3. Review Active Projects.

Every Friday or Sunday:
1. Run Capture Processor.
2. Run Queue Processor.
3. Run Project Health Monitor.
4. Run Weekly Review.

## Core Files

- `Claude.md` = life/project context
- `00_Capture/Capture.md` = raw inbox
- `01_Active/Active Projects.md` = live work
- `05_Queue/Claude Queue.md` = work for Claude
- `09_Dashboards/Home.md` = main dashboard

## Rule

Do not make Reggie sort everything by hand. That is how productivity systems die.
