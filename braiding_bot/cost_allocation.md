# Audit-Safe Cost Allocation and Timekeeping Plan

## Key Compliance Anchors

- **OAFC** is reimbursement-based and requires detailed support for each reimbursement request (including invoices, original itemized receipts, bank/check proof). It explicitly requires PELs, Level-of-Effort documentation/certification for exempt staff, and signed/supervisor-approved timesheets for nonexempt staff submitted with reimbursement requests. (Source: Grants_104_NOFA_FINAL.docx Para 531-537)
- **Federal awards** require that charges for salaries/wages be based on records accurately reflecting work performed and supported by systems of internal control; costs must be adequately documented and cannot be used as cost sharing for other federally financed programs. (Source: 2 CFR 200.403, 200.405, 200.430, 200.302, 200.303)

## Timesheet and LOE Rules (OAFC + Federal-Ready)

### 1. OAFC Reimbursement Support Rules (Non-Negotiable)

- **Nonexempt staff**: Submit signed or electronically approved timesheets (employee + supervisor) with reimbursement requests.
- **Exempt staff** paid on % effort: Retain records meeting TxGMS "Standards for Documentation of Personnel Expenses" OR use monthly Level-of-Effort Certification if required/approved by CPA.
- Submit a **Personnel Eligibility List (PEL)** before starting reimbursable salary/wage activity and update it with invoices when changes occur.

Source: OAFC NOFA requires monthly reimbursement documentation plus PEL/LOE/timesheet rules (Grants_104_NOFA_FINAL.docx Para 531-537).

### 2. Federal-Ready Controls (Required if OJJDP Enters Later)

- Salaries/wages must be supported by records that accurately reflect work performed and are backed by internal controls.
- Budget estimates alone cannot be the final support; after-the-fact adjustments are required for accuracy.
- If staff time is split across funders in a pay period, time must be recorded by fund code + cost objective, not as "admin."

Source: 2 CFR 200.430; 2 CFR 200.403; 2 CFR 200.405.

### 3. Role-Specific Capture Requirements (Labels Only)

| Role Label | Time Capture Rule |
|---|---|
| LCDC | Time only for oversight/compliance/court integration activities and any direct supervision/fidelity work explicitly documented |
| RSPS | Time by CO01-CO07 depending on activity; no "general support" buckets |
| Certified Family Partner | Time by CO04/CO05 with separate home-visit and education logs |
| Peer Supervisor | Time by CO08 (and CO09 only if reporting/evaluation is explicitly within role scope) |
| MHPS | [NEEDED] Must define MHPS duties or do not charge MHPS time to any grant |

## Documentation Checklist (Audit and Monitoring)

### OAFC Reimbursement Package (Monthly Cadence)

- [ ] Reimbursement request form + attestation
- [ ] Invoices
- [ ] Original itemized receipts + proof of payment (check register/bank statement/credit card proof)
- [ ] PEL (current version) if any salary/wage reimbursement is included
- [ ] Exempt staff: Level-of-Effort records or monthly LEC form (if required/approved)
- [ ] Nonexempt staff: timesheets signed/approved (employee + supervisor)

### Program Delivery Proof (Spine Evidence)

- [ ] Referral logs (court/probation source; date)
- [ ] Intake/consent records
- [ ] Screening and literacy records (WRAT-5/TABE)
- [ ] Contact/session notes for 90-day stabilization
- [ ] MOUD referral and appointment confirmation logs
- [ ] Certified Family Partner: education session sign-ins + home visit logs + service linkage log
- [ ] Naloxone inventory ledger + distribution logs + training records
- [ ] YRC continuing care attendance + re-engagement logs
- [ ] Workforce pathway: eligibility tracking + JI-RPS training rosters (18+)

### Subrecipient Monitoring (Critical Once OJJDP Enters)

- [ ] Executed subrecipient agreements (scope, budget, reporting)
- [ ] Monthly subrecipient expenditure reports + supporting docs
- [ ] Site visit notes and corrective action tracking
- [ ] Subrecipient eligibility checks (SAM.gov exclusion checks for federal pass-through)

Source: Federal pass-through entities must monitor subrecipients and verify subrecipients are not excluded/disqualified (per DOJ/OJP guidance referencing 2 CFR 200.332).
