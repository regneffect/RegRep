## Status Definitions

| Status | Definition | Action Required |
|---|---|---|
| **FIRING** | Convergence point is active — channel is producing inputs that demonstrably feed this braiding dimension | None — document evidence |
| **DORMANT** | Convergence point exists in design but has not produced measurable output this quarter | Activate within 30 days or document reason for dormancy |
| **BROKEN** | Convergence point has failed — channel output is not reaching this braiding dimension | Immediate corrective action required; escalate to Program Director |
