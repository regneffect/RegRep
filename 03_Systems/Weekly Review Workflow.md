# Weekly Review Workflow

## Purpose

Review the week, identify what moved, what stalled, what matters next, and what Reggie should focus on.

## Claude Instructions

1. Read `Claude.md`.
2. Read `01_Active/Active Projects.md`.
3. Read `05_Queue/Claude Queue.md`.
4. Read `06_Calendar/Weekly Plan.md`.
5. Read notes created or modified this week.
6. Read generated outputs from this week.
7. Identify:
   - completed work
   - stalled work
   - open loops
   - funding opportunities
   - people waiting on Reggie
   - tasks Claude can help finish
   - next week's top priorities
