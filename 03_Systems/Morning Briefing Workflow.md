# Morning Briefing Workflow

## Purpose

Generate a short morning briefing that tells Reggie what matters most today.

## Claude Instructions

When running this workflow:

1. Read `Claude.md`.
2. Read `01_Active/Active Projects.md`.
3. Read `05_Queue/Claude Queue.md`.
4. Read `06_Calendar/Weekly Plan.md`.
5. Read recent notes in `00_Capture/Capture.md`.
6. Identify the most important work for today.

## Output Location

Save output to:

`04_Generated/Morning Briefings/YYYY-MM-DD Morning Briefing.md`

Create the folder if needed.
